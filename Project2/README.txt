Instructions

Part 1 - Facebook Graph
1) Install dependencies using requirements_facebook.txt
2) Open Jupyter Notebook
3) Using data in data/facebook_combined.txt and run the cells.

Part 2 - Google+
1) Download dataset at: http://snap.stanford.edu/data/egonets-Gplus.html
2) Upload 'Part2-GooglePlus.ipynb' to Google Colab. 
3) Upload Google+ dataset
4) In the second cell, edit the 'DATA_PATH' to the folder containing the dataset
5) Run the cells

Part 3 - Cora
- For Graph Convolutional Networks and Node2Vec:
Upload the notebooks to Google Colab and run the cell.

- For PageRank:
1) Upload the notebook to Google Colab
2) Upload 'cora.tgz' file in the 'data' folder.
3) Run the cells
